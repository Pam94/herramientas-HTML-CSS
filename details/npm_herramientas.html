Gestor de paquetes Npm

Por Pamela Fernández, 14 de abril de 2020

Npm se trata de un gestor de paquetes sirve para instalar y gestionar dependencias externas a nuestro proyecto.

Puede tratarse de una dependencia necesaria para el desarrollo, tales como un optimizador de imágenes, o de una librería que se utilice en el front-end, como por ejemplo Vue, React o Angular.

Una alternativa compatible con npm, aunque con pequeñas diferencias en el funcionamiento, es Yarn.
